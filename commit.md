## git commit

**git commit *[файл]*** - добавляет файл в индекс.

Чтобы добавить все файлы в каталоге в индекс (кроме игнорируемых), используйте команду:

```bash=
git commit -m "название вашего изменения на английском (коротко и локанично"
```
